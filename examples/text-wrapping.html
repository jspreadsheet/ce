Text wrapping

The jExcel default configuration is not wrap the text. But, you can change this behavior by using the wordWrap option. When true, jExcel will keep any new line resizing the height of the related cell.

wordWrap: true Equals css: white-space: pre;

wordWrapType: 'normal' Support all types of white-space style.

To see all white-space style https://developer.mozilla.org/en-US/docs/Web/CSS/white-space
